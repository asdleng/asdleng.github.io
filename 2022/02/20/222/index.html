Veröffentlicht am 2022-02-20

KF

Predict: \[ \hat{x}_t = A_tx_{t-1}+B_tu_t\\ \hat{P}_t = A_tP_{t-1}A_t^T+R_t \]

Update: \[ K_t = \hat{P}_tC^T_t(C_t\hat{P}_tC^T_t+Q_t)^{-1}\\ x_t = \hat{x}_t+K_t(z_t-C_t\hat{x}_t)\\ P_t = (I-K_tC_t)\hat{P}_t \]

EKF

Predict:

\[ \hat{x}_t = f(x_{t-1},u_t)\\ \hat{P}_t = F_tP_{t-1}F_t^T+R_t \] Update: \[ K_t = \hat{P}_tH^T_t(H_t\hat{P}_tH^T_t+Q_t)^{-1}\\ x_t = \hat{x}_t+K_t(z_t-h(\hat{x}_t))\\ P_t = (I-K_tH_t)\hat{P}_t \]

iEKF

Predict: \[ \hat{x}_t = f(x_{t-1},u_t)\\ \hat{P}_t = F_tP_{t-1}F_t^T+R_t \] Update: \[ \hat{x}_t^0 = \hat{x}_t\\ \hat{P}_t^0 = \hat{P}_t\\ \begin{align} for &\space i=1 \space to \space times \space do\\ &K_t^i = \hat{P}^{i-1}_t {H^{i-1}_t}^T (H_t^{i-1}\hat{P}_t^{i-1} {H^{i-1}_t}^T+Q_t)^{-1}\\ &\hat{x}_t^i = \hat{x}_t^{i-1} + K_t^i(z_t-h(\hat{x}_t^{i-1}))\\ &\hat{P}_t^i = (I-K_t^i H_t^{i-1})\hat{P}_t^{i-1}\\ end&for\\ \end{align}\\ x_t = \hat{x}_t^i\\ P_t = \hat{P}_t^i \]
